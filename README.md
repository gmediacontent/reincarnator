# Reincarnator
Dieu a créé l’abîme pour échapper à son ennui. Piégée dans le gouffre infernal, l’humanité était sur le chemin de l’apocalypse…

« Est-ce fini pour nous… maintenant… »

Au bord de l’anéantissement de l’humanité, les quatre survivants les plus forts ont tenté de l’empêcher en remontant le temps jusqu’au moment où l’humanité a été entraînée dans l’abîme. Mais seul l’un d’entre eux a pu retourner dans le passé.

« Qu’est-ce qu’il y a à penser ? N’a-t-on pas déjà décidé qui irait ? Le plus fort d’entre nous, Kang Hansoo. »

Voici l’histoire de la vie du légendaire Kang Hansoo, qui est revenu pour sauver l’humanité de sa perte. Sera-t-il capable de sauver tout le monde ?
